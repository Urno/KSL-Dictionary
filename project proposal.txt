수어 사전 제작
한국의 공용어가 된 한국수어를 배울 수 있는 공식적인 사이트 http://sldict.korean.go.kr/front/main/main.do#의 경우 검색이나 분류가 정확하게 되어 있지 않기 때문에 비장애인들이 찾기 힘든 부분이 많습니다. 따라서 이를 보완할 수 있는 프로그램을 만들어 보고자 합니다. 예를 들면 유사한 단어를 입력했을 때 원하는 단어를 찾을 수 있도록 돕거나, 비슷하거나 헷갈리는 수어를 함께 찾을 수 있도록 하여 검색의 용이성을 높이고자 합니다.

1. 공식 사이트의 단어 목록과 수형을 읽어들일 수 있는 입력 부분을 제작하여 사전을 만든다. 
   (손 모양, 비수지신호, 의미 등의 다양한 정보를 저장하는 데이터 베이스 형태로 예상)

2. 사전을 세부적으로 분류할 수 있는 몇 단계의 기준을 제작합니다. (손 모양, 의미별 분류 등)

3. 사용자가 검색할 수 있는 입력 부분을 제작합니다. (입력 시 2번의 기준을 사용할 수 있도록 활용)
  수어의 특성 상 검색이 어려운 부분이 많은데, 딥러닝 등을 통해 동영상이나 사진을 통한 검색도 가능할 수 있도록 구상 중...

4. 제작된 사전의 정보로부터 사용자가 필요한 정보를 선택적으로 추려내고 결과를 보여줍니다.
   사용자가 원하는 부분을 선택하여 공식 사이트와 연결해서 보여주는 출력 부분을 제작합니다.

5. 추가적인 색인, 사전 추가 기능 등을 통해 새로운 정보를 수용할 수 있는 부분도 고려 중입니다.

문제점
 - 업데이트가 느린 편
 - 검색법이 어려운 경우 多
 - 타 언어 사전에 비해 부족한 기능
 - 수화 인식의 문제

기능 보완
 - 기존 사전의 데이터베이스
 - 개인 사용자의 단어 추가 기능 with 사진, 영상
 - 검색 기능의 다양화(수형, 비수지신호, 분류)
 - 사진, 동영상 속 수화 인식…? by 딥러닝
 - 비장애인을 위한 용어 설명 기능
 - 유사 단어, 반대어, 
 - 사전 출력 기능(사진, 글, 영상)
